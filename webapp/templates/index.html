Sign Edge

Recognition and classification of street signs is important for use in various sorts of visual markets. This is a project to create a machine learning method for classification of the type of street sign that is shown in an image.

The Method:

Images used were from the LISA Dataset, a set of ~6500 images of US based street signs. These images came as labeled captures of dashcam footage, and had to be cropped and resized before being passed through the convolutional neural network. The network used was V3 this is a pre-trained convolutional neural network that was trained on a set of ~10 million images, I use the pre-classification layer from each image to create features to train my own classification model on. Check out the code on my Github.
