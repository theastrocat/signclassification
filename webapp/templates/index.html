Sign Edge

Recognition and classification of street signs is important for use in various sorts of visual markets. This is a project to create a machine learning method for classification of the type of street sign that is shown in a image. The method used will transfer learning from a pre-trained convolutional neural network. This will create a series of features that can be used to fit a machine learning classification model.

The Method:

Images used were from the LISA Dataset, a set of ~6500 images of US based street signs. These images came as labeled captures of dashcam footage, and had to be cropped and resized before being passed through the convolutional neural network.
